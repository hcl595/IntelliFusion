当然,我可以为您编写一段冒泡排序程序。下面是一个简单的程序,可以从用户那里获取输入并输出一条问候消息:

c++
#include  
using namespace std; 
int main() 
{ 
    int arr[100]; 
    cout << "Please enter your name: "; 
    cin >> arr[0]; 
    cout << "Hello, " << arr[0] << "! Nice to meet you." << endl; 
    cout << "And now, the array is: " << endl; 
    for (int i = 1; i < arr.size(); i++) 
        cout << arr[i] << " "; 
    cout << endl; 
    return 0; 
}

运行这个程序后,它会首先向用户询问输入你的名字,然后输出一条问候消息,使用你的名字来定制问候。
